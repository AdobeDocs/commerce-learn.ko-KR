# Commerce MCP 개요

[!BADGE 준비 중]{type=Informative tooltip="이 기능은 곧 제공될 예정이며 변경될 수 있습니다."}

Adobe Commerce용 Commerce MCP는 개발자가 실시간 상거래 데이터 및 도구를 사용하여 검색에서 체크아웃에 이르기까지 사용자 지정 에이전트 경험을 구축할 수 있도록 지원합니다. 이 비디오에서 맞춤형 AI 어시스턴트는 Commerce MCP를 사용하여 구매자 의도를 이해하고 관련 제품 추천을 제공함으로써 대화를 전환합니다.

## 이 비디오는 누구의 것입니까?

* Commerce 스토어 소유자
* 머천다이저
* 디지털 마케터

## 비디오 콘텐츠

* 필터 및 검색 창 대신 자연어를 사용한 대화형 제품 검색
* 장바구니 콘텐츠 및 구매자 환경 설정을 유지하는 컨텍스트 인식 세션 관리
* 페이지 다시 로드나 확인 모델 없이 입력하여 양방향 장바구니를 관리할 수 있습니다.
* 다른 곳으로 이동하지 않고 대화 내에서 체크아웃 완료됨
* 구매한 제품에 대한 구매 후 컨텍스트 기반 지침
