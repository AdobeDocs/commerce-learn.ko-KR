# 새 페이지 만들기

하나의 매개 변수로 json을 반환하는 페이지를 만듭니다.

## 이 비디오 누구?

- 개발자

## 페이지 추가 단계

- 모듈 만들기
- 추가 **route.xml** 파일
- 컨트롤러(작업) 파일 추가

## 모듈을 만드는 단계

- 모듈 폴더 만들기
- 만들기 `etc/module.xml` 파일
- 만들기 `registration.php` 파일
- 를 실행합니다. `bin/magento setup:upgrade` 새 모듈 설치 스크립트
- 모듈이 작동하는지 확인합니다
