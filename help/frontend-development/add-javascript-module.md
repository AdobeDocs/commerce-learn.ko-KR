# JavaScript 모듈 추가

{{only-for-on-prem-commerce-cloud}}

상점 정면에 &quot;Hello World&quot;라는 인사말을 표시하는 간단한 JavaScript 모듈을 개발하는 방법에 대해 알아봅니다.

## 이 비디오는 누구의 것입니까?

* 개발자

## 단계

1. 모듈을 만듭니다.
2. `requirejs-config.js` 및 JavaScript 모듈 파일을 추가합니다.
3. 모듈용 템플릿을 로드하는 레이아웃 업데이트를 만듭니다.
4. 템플릿 파일을 만듭니다.
5. 모듈을 등록하고 상점 첫 화면에서 테스트합니다.

## 비디오 콘텐츠

* 모듈 및 RequireJS 구성을 설정합니다.
* 레이아웃 업데이트 및 템플릿을 통해 스크립트를 연결합니다.
* 페이지에서 인사말을 확인합니다.
