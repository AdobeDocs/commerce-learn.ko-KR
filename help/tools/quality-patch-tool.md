# 품질 패치 도구 사용

품질 패치 도구는 Adobe Commerce 및 Magento Open Source용 품질 패치를 제공하는 명령줄 도구입니다. 이를 통해 다음을 수행할 수 있습니다.

* 최신 품질 패치에 대한 일반 정보 보기
* 품질 패치 적용
* 적용된 패치 되돌리기

Adobe Commerce 지원 팀과 Magento Open Source 커뮤니티는 품질 패치를 개발합니다.

## 이 비디오는 누구의 것입니까?

* 백엔드 개발자
* 사이트 관리자
