# PhpStorm에서 호환성 업그레이드 도구 사용

{{only-for-on-prem-commerce-cloud}}

업그레이드 호환성 도구(UCT)는 현재 버전과 대상 업그레이드 버전 간의 비호환성을 몇 분 안에 분석합니다. PhpStorm 플러그인을 사용하면 IDE에서 쉽게 도구를 실행할 수 있습니다.

이 비디오에서:

* 업그레이드 호환성 도구의 기능
* Magento PhpStorm 플러그인과 함께 사용하는 방법
* 모듈 심각도 및 비호환성 개수별로 HTML 결과를 읽는 방법

## 이 비디오는 누구의 것입니까?

* 개발자

## 비디오 콘텐츠

* PhpStorm에서 UCT를 설치하고 실행합니다.
* HTML 보고서를 검토하고 수정 사항의 우선 순위를 지정합니다.
