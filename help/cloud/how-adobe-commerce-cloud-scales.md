# Adobe Commerce Cloud의 확장 방법

이 튜토리얼에서는 Adobe Commerce Cloud에서 용량이 작동하는 방식, 자동으로 확장되는 방식, Adobe 또는 지원을 포함하는 시기 및 예상치 못한 바이럴 트래픽과 비교하여 계획된 정점에 대해 계획하는 방법에 대해 설명합니다. Experience League 비디오를 녹화할 때 트랜스크립트를 설명 스크립트로 사용합니다.

## 이 비디오는 누구의 것입니까?

* 솔루션 설계자
* 클라우드 운영 엔지니어
* 기술 프로그램 관리자
