# 워크플로우를 사용한 API Mesh 스타터 키트

메쉬를 배포하고 로드 테스트를 수행하기 위해 API Mesh 스타터 키트에 포함된 GitHub 워크플로에 대해 알아봅니다.

## 이 비디오는 누구의 것입니까?

* 프론트엔드 개발자
* 백엔드/API 개발자
* DevOps 엔지니어

## 비디오 콘텐츠

* 메쉬를 자동으로 배포하거나 업데이트하는 deployMesh 워크플로
* K6 부하 테스트 워크플로우를 사용한 부하 테스트
