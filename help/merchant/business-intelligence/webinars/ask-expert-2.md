# Business Intelligence - 전문가에게 문의, 2부

이 기록된 웨비나에서는:

- 공통 테이블 표현식을 사용한 SQL 최적화
- SQL 기반 보고서의 우수 사례
- SQL 기반 보고 데모
- 질문 및 답변

>[!TIP]
>
>Business Intelligence을 처음 사용하는 경우 [시작하기 비디오 시리즈](./../1-overview.md) 는 데이터 집계, 분석 및 시각화를 통해 더 나은 비즈니스 통찰력과 결과를 도출하기 위한 전체 개요를 제공합니다.

## 이 비디오 누구?

- 비즈니스 사용자
- 데이터 분석가
