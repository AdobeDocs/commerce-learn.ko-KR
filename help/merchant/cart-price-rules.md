# 장바구니 가격 규칙 만들기

장바구니 가격 규칙은 일련의 조건을 기준으로 장바구니의 항목에 할인을 적용합니다. 조건이 충족될 때 또는 고객이 유효한 쿠폰 코드를 입력하면 할인이 자동으로 적용됩니다. 적용되면 할인이 소계 아래의 장바구니에 표시됩니다. 장바구니 가격 규칙은 상태와 날짜 범위를 변경하여 시즌 또는 프로모션에 필요에 따라 사용할 수 있습니다.

이 비디오 누구?

- eCommerce 마케터
- 웹 사이트 관리자
