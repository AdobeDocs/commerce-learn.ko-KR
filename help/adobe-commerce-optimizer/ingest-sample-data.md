# 샘플 데이터 수집 Adobe Commerce Optimizer

Adobe Commerce Optimizer 인스턴스에 대한 샘플 데이터를 수집하는 방법을 알아봅니다. 이 비디오에서는 공식 GitHub 저장소 readme 파일을 안내서로 사용하여 시작부터 끝까지 전체 프로세스를 간략하게 설명합니다.

## 이 비디오는 누구의 것입니까?

* Commerce 솔루션 설계자
* 기술 마케팅 엔지니어
* eCommerce 플랫폼 관리자

## 비디오 콘텐츠
