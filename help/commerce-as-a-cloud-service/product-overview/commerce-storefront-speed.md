# Edge Delivery으로 상점 성과 향상

Edge Delivery Services에서 제공하는 Adobe Commerce 상점 이 어떻게 뛰어난 속도와 복원력을 달성하는지 알아보십시오. 지능형 페이지 렌더링, 영구 캐싱 및 CDN 아키텍처가 Lighthouse 점수, Core Web Vitals 및 SEO를 즉시 개선하는 방법에 대해 알아봅니다.

## 이 비디오는 누구의 것입니까?

* 프론트엔드 개발자
* 사이트 성능 엔지니어
* 전자 상거래 관리자

## 비디오 콘텐츠

* Edge Delivery 아키텍처를 통해 Lighthouse에서 상점 점수가 90점을 넘고 Core Web Vitals을 통과하는 방법에 대해 알아봅니다.
* CDN 기반 게재 및 캐싱이 추가 노력 없이 SEO, 트래픽 및 전환율을 개선하는 방법을 알아봅니다.
