# Adobe Commerce as a Cloud Service으로 마이그레이션

Adobe Commerce에서 Adobe Commerce as a Cloud Service으로 이전하는 비즈니스를 위한 유연한 마이그레이션 전략을 제공하는 방법을 알아보십시오. 전체 마이그레이션, 증분 채택 및 Adobe Commerce Optimizer과 같은 파트너사 솔루션에 대해 알아보십시오. 이 모든 솔루션은 중단을 줄이고 가치 실현 시간을 단축하기 위해 설계되었습니다.

## 이 비디오는 누구의 것입니까?

* 전자 상거래 관리자
* 솔루션 설계자
* IT 프로젝트 관리자

## 비디오 콘텐츠

* 전체 마이그레이션, 단계별 채택 및 컴패니언 솔루션을 포함한 마이그레이션 옵션에 대해 알아봅니다.
* Adobe이 App Builder 및 API Mesh를 통해 도구, 파트너 지원 및 확장성을 통해 마이그레이션을 지원하는 방법에 대해 알아봅니다.
