# 내장 실험으로 상점 최적화

Adobe Commerce as a Cloud Service을 통해 비즈니스 팀이 개발자나 타사 도구에 의존하지 않고 A/B 테스트를 실행하고 상점 콘텐츠를 최적화하는 방법에 대해 알아봅니다. 기본 실험 기능을 사용하여 전환 및 참여에 미치는 영향을 측정하는 방법을 알아봅니다.

## 이 비디오는 누구의 것입니까?

* 전자 상거래 관리자
* 디지털 마케팅 전문가
* 전환 최적화 분석가

## 비디오 콘텐츠

* 상점 성능을 개선하기 위해 Adobe Commerce에서 직접 A/B 테스트를 설정하고 실행하는 방법에 대해 알아봅니다.
* 내장된 실험 및 제품 추천이 전환율과 참여를 높이는 데 어떻게 도움이 되는지 알아보십시오.
