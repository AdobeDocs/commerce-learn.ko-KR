# App Builder 및 API Mesh를 사용하여 Adobe Commerce 확장

Adobe Commerce as a Cloud Service이 App Builder 및 API Mesh를 통해 확장성을 지원하는 방법에 대해 알아봅니다. 상점 경험을 중단하지 않고 백엔드 시스템을 통합하고, 워크플로우를 자동화하고, 사용자 정의 앱을 빌드하는 방법에 대해 알아봅니다.

## 이 비디오는 누구의 것입니까?

* 솔루션 설계자
* 백엔드 개발자
* 통합 전문가

## 비디오 콘텐츠

* App Builder을 통해 Adobe Commerce에 대한 맞춤형 앱 개발 및 워크플로 자동화를 구현하는 방법에 대해 알아봅니다.
* API Mesh가 통합 API 계층을 통해 여러 시스템에서 데이터를 오케스트레이션하는 방법을 살펴봅니다.
