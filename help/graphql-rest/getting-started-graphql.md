# Adobe Commerce용 GraphQL 시작

GraphQLAdobe 을 처음 사용하는 경우 함께 제공되는 자습서를 살펴보기 전에 이 소개 비디오부터 시작하는 것이 좋습니다.

## 이 비디오는 누구의 것입니까?

* Adobe Commerce 또는 을 처음 사용하는 개발자 [!DNL Magento Open Source] GraphQL에 대한 경험이 제한되어 있습니다.

## 비디오 콘텐츠

* GraphQL 소개
* Altair 플러그인 사용
* GraphQL 쿼리
* GraphQL 돌연변이
* GraphQL 스키마
