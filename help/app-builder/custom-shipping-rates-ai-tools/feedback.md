# AI 도구를 사용하여 맞춤형 배송비 만들기 - 피드백 제공

AI 도구를 사용하여 사용자 정의 배송 요금을 생성하는 방법에 대해 알아보십시오. 이 튜토리얼에서는 피드백 프로세스를 안내하여 워크플로를 간소화하고 효율성을 향상시킬 수 있습니다. 피드백을 입력하고 효과적으로 대화를 압축하여 MCP 도구를 통해 원활한 경험을 보장하는 방법을 알아봅니다.

## 이 비디오는 누구의 것입니까?

* 기술 설계자
* 수석 개발자 또는 엔지니어
* 제품 관리자

## 비디오 콘텐츠

* MCP 피드백 도구 작동 방식 알아보기
* IDE에서 Adobe에 피드백을 제공하는 방법을 이해합니다.
