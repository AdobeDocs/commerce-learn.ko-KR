# AI 도구를 사용하여 맞춤형 배송률 만들기 - 기능 구축

Adobe의 AI 기반 도구를 활용하여 개발 워크플로를 간소화하고 체크아웃 기능을 확장하는 방법을 살펴보십시오. 이 튜토리얼에서는 API 키로 보호되는 모의 API를 사용하여 외부 시스템에서 사용자 정의 배송 속도를 통합하는 방법에 대해 설명합니다. 서비스 URL을 구성하고, 보안을 관리하고, 고급 체크아웃 기능을 활성화하는 방법에 대해 알아봅니다. 이 포괄적인 안내서는 AI가 요구 사항 수집에서 구현 계획에 이르기까지 소프트웨어 개발을 가속화하는 방법을 보여줍니다. 자세한 연습은 포함된 비디오를 시청하십시오.

## 이 비디오는 누구의 것입니까?

* 기술 및 솔루션 설계자
* 백엔드 개발자 및 엔지니어
* 구현 엔지니어 및 기술 컨설턴트

## 비디오 콘텐츠

* AI 에이전트는 컨텍스트를 분석하고 역할 기반 기술을 사용하여 개발을 안내합니다.
* 스타터 키트는 보다 빠른 확장 구축을 위한 구조 및 샘플 패턴을 제공합니다.
* MCP 도구는 에이전트가 코드를 계획하고 생성하는 데 사용하는 명령 및 데이터를 노출합니다.
* AI는 요구 사항, 아키텍처 및 구현 결정을 자동화합니다.
