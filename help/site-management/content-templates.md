# 콘텐츠 템플릿 만들기

템플릿을 사용하면 콘텐츠를 만들거나 이전 콘텐츠를 바꿀 때 시간과 노력을 절약할 수 있습니다. 예를 들어 기존 페이지 빌더 콘텐츠를 템플릿으로 저장한 다음 해당 템플릿(모든 콘텐츠 및 레이아웃과 함께)을 다른 영역에 적용하여 페이지 빌더 콘텐츠를 빠르게 만들 수 있습니다.

## 이 비디오는 누구의 것입니까?

- 웹 사이트 관리자
- 콘텐츠 디자이너
