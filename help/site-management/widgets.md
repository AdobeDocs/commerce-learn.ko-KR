# 위젯 만들기

위젯은 저장소에서 콘텐츠를 자동으로 업데이트하도록 쉽게 만들고 수정할 수 있는 재사용 가능한 구성 요소입니다. 이 비디오에서는 위젯을 만들고 관리하는 방법을 알아봅니다.

## 이 비디오 누구?

- 웹 사이트 관리자
- 콘텐츠 디자이너
