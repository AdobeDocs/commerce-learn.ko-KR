# SEO에 대한 URL 재작성 관리

URL 재작성 도구를 사용하면 제품, 카테고리 또는 CMS 페이지와 연결된 모든 URL을 변경할 수 있습니다. 다시 쓰기가 적용되면 이전 URL을 가리키는 모든 링크가 새 주소로 리디렉션됩니다.

## 이 비디오는 누구의 것입니까?

- 웹 사이트 관리자
- 전자 상거래 머천다이저
- eCommerce 마케터
