# 설정 및 구성

표준화된 상관 관계 원격 분석을 사용하여 Adobe Commerce 및 사용자 지정 애플리케이션에서 전체적인 가시성을 구현하는 방법에 대해 알아봅니다. 이 비디오에서는 상관 관계 및 표준화가 중요한 이유에 대해 설명하고 Commerce Observability Module 및 IO Live 원격 분석 라이브러리를 소개합니다. 이러한 도구를 통해 원활한 작동 흐름을 위한 통합 모니터링, 로깅 및 추적을 활성화하는 방법에 대해 알아봅니다.

## 이 비디오는 누구의 것입니까?

* Adobe Commerce 개발자
* 솔루션 설계자
* 기술 제품 관리자

## 비디오 콘텐츠

* 전체 가시성을 위해 원격 분석 표준화 및 상관 관계 분석
* 자동화된 신호에 Commerce Observability Module 사용
* IO Live 원격 분석 라이브러리로 모니터링 확장
