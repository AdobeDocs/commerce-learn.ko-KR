# 종속성 삽입 예

종속성 주입은 개체 A가 이러한 종속성을 제공하는 외부 개체 B에 대한 종속성을 선언할 수 있도록 하는 디자인 패턴입니다. A로 선언된 종속성은 일반적으로 클래스 인터페이스이며 B가 제공하는 종속성은 해당 인터페이스에 대한 구체적인 구현입니다.
